# Citrux Companion 🍊

Citrux Companion 是一款與 [Citrux CLI](https://github.com/sivahuang77/citrux-cli) 完美配對的 VS Code 擴充功能。它支援 VS Code 及其相關分支版本，為您的開發流程提供深度整合的 AI 輔助。

# 核心功能

- **開啟檔案上下文 (File Context)**：Citrux CLI 能即時感知您正在編輯的檔案，從而更精確地理解專案結構與內容。
- **選取範圍上下文 (Selection Context)**：輕鬆存取游標位置或選取的程式碼片段，直接從您的工作中獲取關鍵資訊。
- **原生差異比對 (Native Diffing)**：在編輯器中直接預覽、修改並採納 Citrux CLI 建議的程式碼變更。
- **快速啟動**：透過指令面板 (Cmd+Shift+P / Ctrl+Shift+P) 執行 "Citrux CLI: Run"，即可秒速開啟新的 AI 階段。

# 系統需求

若要使用此擴充功能，您需要：

- VS Code 版本 1.99.0 或更高
- 已安裝 [Citrux CLI](https://github.com/sivahuang77/citrux-cli) 並在整合終端機中執行
