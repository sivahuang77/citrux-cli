## 🌟 為什麼選擇 Citrux CLI？

1.  **完全的原生體驗**：Citrux 從零開始針對終端機（TUI）進行優化，提供流暢的流式輸出與直觀的交互式對話框。
2.  **不綁定模型 (Model Agnostic)**：自由切換 OpenAI, DeepSeek,
    Google 或本地 Ollama 模型，永遠使用最強大的引擎。
3.  **深度上下文感知**：透過 `CITRUX.md`
    協議，AI 真正讀懂您的專案架構與開發規範。
4.  **隱私第一**：您的代碼與 API 金鑰永遠留在本地 `~/.citrux` 目錄中。
